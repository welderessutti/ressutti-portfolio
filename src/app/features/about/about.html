About

Full-stack software developer building reliable applications with Angular, Java and Spring.

I design and develop web applications with a focus on clear architecture, maintainable code and dependable behaviour. My work covers front-end experiences, back-end services and the integration between them, always aiming for solutions that can evolve without unnecessary complexity.

From analytical rigour to software engineering

Professional journey

Before moving into software development, I spent 11 years working in biomedicine and managing a clinical laboratory. That environment required analytical thinking, responsibility, attention to processes, leadership and consistent decision-making.

I brought those principles into technology. After completing a degree in Systems Analysis and Development and postgraduate studies in Java Architecture and Development, I focused my work on full-stack applications using Angular, Java and Spring. Today, I approach software as a system that should remain understandable, testable and reliable as its requirements evolve.
